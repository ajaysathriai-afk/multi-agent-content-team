# 🤖 Multi-Agent Content Team

AI-powered content creation system with debate, research, writing, and editing agents.

## Features

- 🧠 **Debate Agent** - Analyzes topic from multiple perspectives
- 🔎 **Research Agent** - Real-time web search with Tavily API
- ✍️ **Writer Agent** - SEO-optimized article generation
- 🧹 **Editor Agent** - Grammar and style polishing
- 📚 **Knowledge Base** - Stores research for reuse
- 📄 **Markdown Export** - Download publication-ready articles

## Tech Stack

- **CrewAI** - Multi-agent orchestration
- **GPT-4** - Language model
- **Tavily API** - Web research
- **Gradio** - Web interface
- **Python 3.11**

## How It Works

1. **Debate Agent** analyzes the topic from multiple angles
2. **Research Agent** searches the web using Tavily API
3. **Writer Agent** creates SEO-optimized content
4. **Editor Agent** polishes the final article
5. Download as Markdown!

## Setup

Required environment variables:
- `OPENAI_API_KEY` - Your OpenAI API key
- `TAVILY_API_KEY` - Your Tavily API key
